# edit-image

## 이미지 편집 및 그림판 기능
HTML Canvas와 바닐라 javascript로 구현한 이미지 편집기 및 그림판입니다. 
최대 800*800px 크기의 캔버스 내에 그림을 그리거나, 텍스트 추가할 수 있으며, 로컬에 저장되어있는 이미지를 불러 회전시킬 수 있습니다.
